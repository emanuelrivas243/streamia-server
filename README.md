🎬 Streamia - Backend

Backend de una plataforma de streaming de películas, construido con Node.js, TypeScript y Express, conectado a MongoDB y desplegado en Render.

🚀 Tecnologías principales

Node.js + TypeScript

Express.js

MongoDB + Mongoose

JWT (autenticación)

Zod (validaciones)

Rate limiting

SendGrid / Nodemailer (emails)

Render (deployment)


⚙️ Requisitos previos

Asegúrate de tener instalado:

Node.js v16+

MongoDB Atlas o instancia local

Cuenta SendGrid u otro SMTP (opcional)

📦 Instalación

Clonar el repositorio:

git clone <URL-del-repo>
cd streamia-server


Instalar dependencias:

npm install

▶️ Modo desarrollo
npm run dev
